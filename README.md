# Сложно сосредоточиться

Статическая адаптивная страница о причинах проблем с концентрацией и способах вернуть внимание. Проект свёрстан по макету и поддерживает несколько цветовых схем.

Репозиторий: https://github.com/b1abb/slozhno-sosredotochitsya-fd

## Структура проекта

- `index.html` — основная HTML-разметка страницы
- `styles/` — CSS-файлы, переменные и стили цветовых тем
- `scripts/script.js` — логика переключения темы
- `images/` — изображения, favicon и фоновые картинки
- `fonts/` — локальные файлы шрифта IBM Plex Mono
